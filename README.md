# Personal site

A minimal, static portfolio site — plain HTML & CSS, no build step.

## Editing

- `index.html` — page content
- `styles/main.css` — all styling

Open `index.html` in a browser to preview locally.

## Deploy (GitHub Pages)

This site is served as static files. With GitHub Pages enabled on the
`main` branch (root), the site goes live automatically on every push.
